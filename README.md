# Ruang programmer phase 1
Membahas materi mengenai Ionicframework with Angular

## Step By Step
* Clone projek
```bash
git clone https://github.com/firhanhardiansyah/ruangprogrammer1.git
```
* Masuk ke direktori projek
```bash
cd ./ruangprogrammer1
```
* Install npm module
```bash
npm install
```
* Jalankan programnya, Buka terminal
```bash
ionic serve
```
